## Introduction

Amap (`@partme.ai/openclaw-amap`) is an [OpenClaw](https://github.com/openclaw/openclaw) channel plugin integrating the Amap (Gaode) Open Platform Web Service API.

### Core capabilities

- `amap_query_poi` — POI keyword search
- `amap_query_around` — Nearby search
- `amap_place_detail` — Place detail query

## Quick start

```bash
openclaw plugins install @partme.ai/openclaw-amap
```
