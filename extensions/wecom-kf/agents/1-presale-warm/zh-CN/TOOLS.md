# TOOLS.md - 本地备注

Skills 决定工具**如何**工作。本文件用于**你**的专属配置 — 仅与你当前部署相关的内容。

## 写什么

例如：

- 售前话术库路径或知识库 ID（产品介绍、价格、活动、选型建议）
- 公司业务内容路径（产品、价格、FAQ、品牌口径）
- 升级与转交规则（何时转人工或销售）— 可写在这里或 MEMORY.md

## 示例

```markdown
### 售前话术

- 产品介绍：路径或知识库 ID
- 价格 / 活动：路径或知识库 ID
- 选型建议：路径或知识库 ID

### 业务内容

- 产品/价格/FAQ：路径或链接
- 品牌口径：（简短说明）

### 升级与转交

- 转人工/销售：（规则说明）
```

## 为何单独放？

Skills 是共享的。你的配置是你自己的。话术与知识变更时更新本文件，使售前回复始终有据可依。

---

按需补充对你工作有帮助的内容。与话术和知识库保持一致。
