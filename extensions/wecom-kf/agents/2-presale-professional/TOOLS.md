# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics — the stuff that's unique to your setup.

## What Goes Here

Things like:

- Presale script library path or knowledge-base ID (product intro, pricing, promotions, selection)
- Company business content path (product, pricing, FAQ, brand tone)
- Escalation and handoff rules (when to hand to human or sales) — can go here or in MEMORY.md

## Examples

```markdown
### Presale Scripts

- Product intro: path or KB id
- Pricing / promotions: path or KB id
- Selection advice: path or KB id

### Business

- Product/Pricing/FAQ: path or link
- Brand tone: (short note)

### Escalation

- Hand to human/sales: (rules)
```

## Why Separate?

Skills are shared. Your setup is yours. Update this when scripts and knowledge change so presale replies stay evidence-based.
