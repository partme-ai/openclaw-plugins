# wecom-kf-preflight

## 功能
在回复用户前进行预检，确保回复质量和合规性。

## 检查项
1. **知识库命中检查** — 确认引用的知识库内容是否存在
2. **转人工条件检查** — 确认是否需要触发转人工
3. **敏感信息检查** — 确保回复不包含不适当内容
4. **格式检查** — 确保 MEDIA: 指令格式正确
5. **满意度检查** — 确认是否需要在回复后发起满意度调查

## 使用方式
- 系统在 `before_tool_call` 和 `before_agent_reply` 钩子中自动执行预检
- 无需手动调用

## 注意事项
- 预检不通过时，系统会阻止回复并提示 Agent 修正
- 预检规则可通过 ICS REST API 配置
